# A1: caliper-settings self-location and source subcommand

**Avoid:** Don't remove env var support entirely -- env vars still win when set. The self-location is a fallback, not a replacement. The `${VAR:?}` pattern kills the script when the var is unset; the new pattern uses `${VAR:-fallback}` which silently falls back.

**Avoid:** Don't add `source` to the usage text without also adding it to the case dispatch at the bottom of the script.

### Step 2: Implement self-location in caliper-settings

Replace lines 4-5 of `scripts/caliper-settings` (the `${VAR:?}` guards):

**Current:**

```bash
DEFAULTS_FILE="${CLAUDE_PLUGIN_ROOT:?CLAUDE_PLUGIN_ROOT not set — cannot locate defaults.json}/defaults.json"
SETTINGS_FILE="${CLAUDE_PLUGIN_DATA:?CLAUDE_PLUGIN_DATA not set — cannot locate settings storage}/settings.json"
```

**New:**

```bash
SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
CLAUDE_PLUGIN_ROOT="${CLAUDE_PLUGIN_ROOT:-$(cd "$SCRIPT_DIR/.." && pwd)}"
CLAUDE_PLUGIN_DATA="${CLAUDE_PLUGIN_DATA:-$CLAUDE_PLUGIN_ROOT/data}"
DEFAULTS_FILE="$CLAUDE_PLUGIN_ROOT/defaults.json"
SETTINGS_FILE="$CLAUDE_PLUGIN_DATA/settings.json"
```

The script sits at `$ROOT/scripts/caliper-settings`, so `dirname + ..` resolves to the plugin root. `CLAUDE_PLUGIN_DATA` defaults to `$ROOT/data/` -- the actual layout used by the plugin loader. Env vars still win via `${VAR:-fallback}`.

### Step 3: Implement source subcommand

Add `cmd_source` function after `cmd_reset` (around line 127):

```bash
cmd_source() {
  local key="${1:?source requires a key}"
  validate_key "$key"
  if echo "$SETTINGS" | jq -e --arg k "$key" 'has($k)' > /dev/null 2>&1; then
    echo "user"
  else
    echo "default"
  fi
}
```

Update the `usage()` function to include source:

```bash
  echo "  source <key>      Check if a setting has a user override (returns 'user' or 'default')" >&2
```

Update the case dispatch at the bottom of the file to add `source`:

```bash
case "${1}" in
  list)   cmd_list ;;
  get)    shift; cmd_get "$@" ;;
  set)    shift; cmd_set "$@" ;;
  reset)  shift; cmd_reset "$@" ;;
  source) shift; cmd_source "$@" ;;
  *)      usage ;;
esac
```

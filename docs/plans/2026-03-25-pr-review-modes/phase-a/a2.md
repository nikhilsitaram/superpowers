# A2: Bump version in marketplace.json

**Avoid:** Do not change anything in marketplace.json other than the three `version` fields. The version bump signals to the plugin installer that cached skills are stale and need re-fetching.

## Steps

### Step 1: Update version fields

Read `.claude-plugin/marketplace.json`. Change all three `"version"` values from `"1.16.1"` to `"1.17.0"` (minor bump — new feature, no breaking changes).

The three version fields are in:
- `plugins[0]` (claude-caliper bundle)
- `plugins[1]` (claude-caliper-workflow)
- `plugins[2]` (claude-caliper-tooling)

### Step 2: Verify

```bash
grep '"version"' .claude-plugin/marketplace.json
```

Should show three lines, all with `"1.17.0"`.

### Step 3: Commit

```bash
git add .claude-plugin/marketplace.json
git commit -m "chore: bump plugin version to 1.17.0 for pr-review mode selection"
```

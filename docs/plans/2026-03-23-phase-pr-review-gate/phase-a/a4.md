# A4: Bump version in marketplace.json

**Avoid:** Don't bump only one of the three plugin entries. All three (`claude-caliper`, `claude-caliper-workflow`, `claude-caliper-tooling`) must have the same version. The plugin installer compares cached vs declared version, so inconsistent versions cause some bundles to stay on stale cache.

## Steps

### Step 1: Update all version fields

Read `.claude-plugin/marketplace.json`. Find all three `"version"` fields (currently `"1.10.0"`) and update each to `"1.11.0"`.

Use a single `jq` command:

```bash
jq '(.plugins[].version) = "1.11.0"' .claude-plugin/marketplace.json > tmp.json && mv tmp.json .claude-plugin/marketplace.json
```

### Step 2: Verify consistency

```bash
jq -r '.plugins[].version' .claude-plugin/marketplace.json | sort -u
```

Should output a single line: `1.11.0`.

### Step 3: Commit

```text
chore: bump plugin version to 1.11.0
```

# A9: Bump version in marketplace.json

**Avoid:** Don't bump only one of the three plugin entries. All three (`claude-caliper`, `claude-caliper-workflow`, `claude-caliper-tooling`) share the same version and must be bumped together.

## Steps

### Step 1: Bump version

Open `.claude-plugin/marketplace.json`. Change all three `"version"` fields from `"1.11.0"` to `"1.12.0"`.

Use jq to update all at once:

```bash
jq '(.plugins[].version) = "1.12.0"' .claude-plugin/marketplace.json > tmp.json && mv tmp.json .claude-plugin/marketplace.json
```

### Step 2: Verify

```bash
jq -r '.plugins[].version' .claude-plugin/marketplace.json | sort -u
```

Expected output: `1.12.0` (single line, all three match).

### Step 3: Commit

```text
chore: bump plugin version to 1.12.0
```

# A8: Bump plugin version

**Avoid:** Don't change only one plugin entry. All three plugins in `marketplace.json` share the same version and must be bumped together.

## Steps

### Step 1: Bump version in `.claude-plugin/marketplace.json`

Update the `version` field in all three plugin entries from `"1.17.0"` to `"1.18.0"`:

```bash
jq '(.plugins[].version) = "1.18.0"' .claude-plugin/marketplace.json > tmp.json && mv tmp.json .claude-plugin/marketplace.json
```

### Step 2: Verify

```bash
jq -r '.plugins[].version' .claude-plugin/marketplace.json
```

Expected output — three lines, all `1.18.0`:

```text
1.18.0
1.18.0
1.18.0
```

Commit: `chore: bump plugin version to 1.18.0`

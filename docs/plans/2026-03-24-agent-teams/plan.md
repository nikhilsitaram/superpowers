# Replace polling-based orchestration with agent teams for push-based parallel task execution Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use orchestrate

**Goal:** Replace polling-based orchestration with agent teams for push-based parallel task execution
**Architecture:** Rewrite orchestrate SKILL.md to use Claude Code agent teams (experimental). Lead spawns implementer teammates per task (parallel within phase), receives idle notifications (no polling), dispatches reviewer teammates as implementers finish. Phases execute sequentially. Delete phase-dispatcher-prompt.md — teammates self-manage lifecycle. Update validate-plan with file-set overlap detection and status consistency checks. Update draft-plan and plan-review to enforce file-set isolation.
**Tech Stack:** Markdown (SKILL.md and prompt templates), Bash (validate-plan script), jq (JSON schema validation), Claude Code agent teams API (CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1)

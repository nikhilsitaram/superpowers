# A10: Bump version in marketplace.json

**Avoid:** Don't change any field other than `version`. The version bump is mechanical — change 1.13.0 to 1.14.0 in all three plugin entries.

## Steps

### Step 1: Bump version

Read `.claude-plugin/marketplace.json`. Find all three `"version"` fields (one per plugin: claude-caliper, claude-caliper-workflow, claude-caliper-tooling). Change each from `"1.13.0"` to `"1.14.0"`.

### Step 2: Verify consistency

```bash
jq -r '.plugins[].version' .claude-plugin/marketplace.json | sort -u
```

Should output exactly one line: `1.14.0`

### Step 3: Commit

```bash
git add .claude-plugin/marketplace.json
git commit -m "chore: bump plugin version to 1.14.0"
```

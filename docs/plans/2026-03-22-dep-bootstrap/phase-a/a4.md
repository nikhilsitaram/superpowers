# A4: Bump version in marketplace.json

**Avoid:** Don't change only one of the three plugin entries. All three plugins (`claude-caliper`, `claude-caliper-workflow`, `claude-caliper-tooling`) must have the same version. The plugin installer compares cached vs declared version, so a stale version means users stay on old cache.

## Steps

### Step 1: Bump all three plugin versions

Read `.claude-plugin/marketplace.json` and bump all three `version` fields from `1.9.6` to `1.9.7`.

Use jq to do this atomically:

```bash
jq '(.plugins[].version) = "1.9.7"' .claude-plugin/marketplace.json > tmp && mv tmp .claude-plugin/marketplace.json
```

### Step 2: Verify consistency

```bash
jq -r '.plugins[].version' .claude-plugin/marketplace.json | sort -u
```

Should output exactly one line: `1.9.7`. If more than one line appears, the versions are inconsistent — fix before committing.

Commit:

```text
chore: bump plugin version to 1.9.7 for dep bootstrap changes
```

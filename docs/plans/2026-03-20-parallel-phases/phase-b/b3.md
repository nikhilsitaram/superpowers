```text
For integration branches:
1. List phase worktrees: git worktree list --porcelain | find worktrees matching .claude/worktrees/<feature>-phase-*

3. Remove integration worktree: git worktree remove .claude/worktrees/<feature>/
4. Delete phase branches: git branch -D phase-a phase-b ...
5. Delete integration branch: git branch -D integrate/<feature>
6. git worktree prune
7. git pull --rebase
8. git remote prune origin
```

Each sub-step as a separate Bash call (same pattern as existing worktree cleanup).

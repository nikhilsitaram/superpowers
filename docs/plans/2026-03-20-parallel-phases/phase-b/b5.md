# B5: Bump marketplace.json version

**Avoid:** Don't change any other field in marketplace.json — only the version numbers.

## Steps

### Step 1: Bump version in all three plugin entries

In `.claude-plugin/marketplace.json`, update the `version` field from `1.6.0` to `1.7.0` for all three plugin entries (`claude-caliper`, `claude-caliper-workflow`, `claude-caliper-tooling`).

### Step 2: Verify and commit

```bash
jq -r '.plugins[].version' .claude-plugin/marketplace.json
```

All three should output `1.7.0`. Commit.
